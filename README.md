  <h2>What These Codes Help With</h2>
  <ul>
    <li><strong>Pre-Processing:</strong> Creating and managing amplitude data and other inputs required before running ABAQUS analysis.</li>
    <li><strong>Input File Development:</strong> Assisting in writing and modifying ABAQUS <code>.inp</code> files programmatically.</li>
    <li><strong>LET Theory Implementation:</strong> Scripts related to calculations and data handling based on Layered Elastic Theory concepts used in the course.</li>
    <li><strong>Post-Processing:</strong> Reading ABAQUS output data and generating plots and organized results.</li>
    <li><strong>Example Files:</strong> Sample data used during demonstrations in the course.</li>
  </ul>

  <h2>How to Use</h2>
  <ol>
    <li>Run your ABAQUS flexible pavement model as demonstrated in the course.</li>
    <li>Use the relevant script from this repository depending on your task (pre-processing, input editing, LET calculations, or post-processing).</li>
    <li>Place your ABAQUS output or input files alongside the script as needed.</li>
    <li>Run the Python script and follow the instructions inside the code.</li>
  </ol>
